=== SSL Cert Installation ===
Add repository
  vi /etc/apt/sources.list
  deb http://ftp.debian.org/debian jessie-backports main

Update and install
  apt-get update
  apt-get install python-certbot-apache -t jessie-backports
  certbot --apache -d bigbank.com.br

== Ionic == 
=== Install NodeJs ===
  sudo apt-get update
  sudo apt-get upgrade
  sudo apt-get install curl
  curl -sL https://deb.nodesource.com/setup_6.x -o nodesource_setup.sh
  sudo bash nodesource_setup.sh
  sudo apt-get install nodejs

=== Install Ionic & Cordova ===
  sudo npm install -g cordova ionic

* You may need
  sudo apt-get install libc6:i386 libncurses5:i386 libstdc++6:i386 lib32z1 libbz2-1.0:i386
  sudo ln -s /usr/bin/nodejs /usr/bin/node

=== Test ===
  ionic start myApp tabs
  cd myApp/
  ionic serve


== Codiad ==

=== Install ===
  sudo apt-get install git
  sudo git clone https://github.com/Codiad/Codiad /var/www/html/codiad
  sudo touch /var/www/html/codiad/config.php
  sudo chown www-data:www-data -R /var/www/html/codiad

Open your installation with http://your-ip-adress/codiad and finish the installation.

=== Config a new project ===
After finished the installation (create a html project)

  sudo mkdir /var/www/html
  sudo rm -rf /var/www/html/codiad/workspace/html
  sudo ln -s /var/www/html /var/www/html/codiad/workspace/html
  sudo chown www-data:www-data -R /var/www/html

== RSS ==
=== Install mcrypt ===
  sudo php5enmod mcrypt
  service apache2 restart

=== Get TTRSS ===
  sudo apt-get install git
  cd /var/www/html/
  git clone https://tt-rss.org/git/tt-rss.git rss

=== Database ===

* Create database http://*.bigbank.com.br/phpmyadmin

* Create user rss into the database

* Restore database

  mysql -u rss -p rss < file.sql

=== Permissions configuration ===
  cd rss/
  chmod -R 777 cache/images
  chmod -R 777 cache/upload
  chmod -R 777 cache/export
  chmod -R 777 cache/js
  chmod -R 777 feed-icons
  chmod -R 777 lock

* Continue on http://*.bigbank.com.br/rss

=== Cron ===
* Configurar o cron para que atualize o status das noticias (usuário não root)
  crontab -e
  */30 * * * * /usr/bin/php /var/www/html/rss/update.php --feeds --quiet

* Reiniciar o cron
  sudo service cron restart



Run 'dpkg-reconfigure tzdata' if you wish to change the timezone.

== Server Monitor Installation (SRVMON) ==

=== Download and copy to the server ===
  http://www.phpservermonitor.org/download/

* Unzip on: /var/www/html 
  tar xvzf phpservermon-3.1.1.tar.gz

* You may need that
  sudo apt-get install php5-curl
  sudo service apache2 restart

* Configurar o Timezone do PHP
  Usando o TimeZone de aqui: http://php.net/manual/en/timezones.america.php
  sudo nano /etc/php5/apache2/php.ini
  sudo service apache2 restart

* Criar o banco de dados no http://*.bigbank.com.br/phpmyadmin

=== Config ===
* Criar usuário srvmon e dar acesso ao banco

* Seguir o passo a passo do http://*.bigbank.com.br/phpmyadmin

* Configurar o cron para que atualize o status dos servidores
  -- Incluir o update automático no cron
  crontab -e
  */10 * * * * root /usr/bin/php /var/www/html/srvmon/cron/status.cron.php

* Reiniciar o cron
  sudo service cron restart
